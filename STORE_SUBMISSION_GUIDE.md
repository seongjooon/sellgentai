🚀 MarginScan - 로켓그로스 마진 계산기

📦 자동 상품 정보 스크래핑

- 부가세 포함 총 수수료 산출

🎯 목표 마진율 기반 분석
- 원하는 마진율 설정 (0~50%)
- 목표 달성을 위한 최대 사입가 추천
- 실시간 수익성 분석

📊 상세한 마진 분석
- 예상 순이익 계산
- 마진율 퍼센티지
- 총 비용 분석

⚠️ 스마트 알림
- 쿠팡 직매입 상품 자동 감지
- 수익성에 따른 색상 코딩

🔒 개인정보 보호
- 모든 계산은 브라우저에서만 수행
- 외부 서버로 데이터 전송 없음
- 사용자 추적 없음

📱 사용 방법
1. 쿠팡 상품 페이지 방문
2. 확장 프로그램 아이콘 클릭
3. 상품 크기 선택 및 원가 입력
4. 실시간 마진 확인

💡 이런 분들께 추천합니다
- 쿠팡 로켓그로스 셀러
- 온라인 리셀러
- 이커머스 사업자
- 상품 소싱 담당자

⚡ 빠르고 정확한 의사결정을 도와드립니다!

## 🔒 개인정보 및 보안

### 개인정보 처리방침
- URL: GitHub Pages에 호스팅하거나 별도 웹사이트 필요
- 파일: `PRIVACY_POLICY.md` 참조

### 권한 사용 정당화

**storage:**
```
목표 마진율 설정을 저장하기 위해 브라우저의 로컬 저장소를 사용합니다.
동기화되지 않으며, 외부로 전송되지 않습니다.
```

**activeTab:**
```
현재 쿠팡 상품 페이지의 정보(가격, 카테고리 등)를 읽어 마진을 계산하기 위해 사용됩니다.
데이터는 저장되거나 전송되지 않습니다.
```

**host_permissions (*.coupang.com):**
```
쿠팡 상품 페이지에서만 작동하도록 제한합니다.
상품 정보를 읽어 실시간으로 마진을 계산하는 용도로만 사용됩니다.
```

## 📦 빌드 및 제출

### 1. 빌드
```bash
npm run build
npm run zip
```

생성된 파일: `.output/marginscan-1.0.0-chrome.zip`

### 2. 테스트
- 로컬에서 충분히 테스트
- 다양한 상품 페이지에서 동작 확인
- 에러 핸들링 확인
- 권한 최소화 확인

### 3. Chrome Developer Dashboard
1. https://chrome.google.com/webstore/devconsole 접속
2. 개발자 계정 등록 ($5 일회성 비용)
3. "새 항목" 클릭
4. ZIP 파일 업로드
5. 스토어 등록 정보 입력
6. 스크린샷 업로드
7. 개인정보 처리방침 URL 입력
8. 검토 제출

### 4. 검토 대기
- 일반적으로 1-3일 소요
- 거부되면 피드백 확인 후 수정
- 재제출 가능
