개인정보 처리방침 (Privacy Policy)

최종 수정일: 2025년 12월

1. 개요

Sellgent AI는 온라인 셀러를 위한 소싱 상품 분석 확장 프로그램입니다. 본 정책은 사용자의 개인정보 보호를 최우선으로 합니다.

2. 수집하는 정보

Sellgent AI는 개인정보를 수집하지 않습니다.

로컬 저장 데이터

  • 사용자 설정: 목표 마진율, 필터 설정, 저장된 상품 목록 등은 브라우저의 localStorage에만 저장됩니다.
  • 계산 데이터: 모든 마진 계산, 가격 분석은 사용자의 브라우저에서만 수행되며 외부로 전송되지 않습니다.
  • 상품 정보: 상품 가격, 카테고리, 이미지 등의 메타데이터는 로컬에서만 처리됩니다.

3. 데이터 사용 방식

데이터 저장

  • 모든 데이터는 사용자의 로컬 브라우저에만 저장됩니다.
  • 어떠한 데이터도 외부 서버로 전송되지 않습니다.
  • 확장 프로그램 제거 시 로컬 데이터는 자동으로 삭제됩니다.

데이터 처리

  • 모든 상품 분석 및 마진 계산은 브라우저에서 실시간으로 수행됩니다.
  • 사용자 추적, 분석, 광고 목적의 데이터 수집을 하지 않습니다.

4. 확장 프로그램 권한 설명

storage

  • 목표 마진율, 필터 설정, 사용자 환경설정을 저장하기 위해 사용됩니다.
  • 로컬 저장소만 사용하며 클라우드 동기화는 없습니다.

activeTab

  • 쿠팡, 아마존 등 쇼핑몰 상품 페이지의 정보를 읽기 위해 사용됩니다.
  • 페이지 내용은 분석 목적으로만 사용되며 저장되거나 외부로 전송되지 않습니다.

host_permissions (쿠팡, 아마존 등)

  • 지원하는 쇼핑몰 상품 페이지에서만 작동하도록 제한됩니다.
  • 상품 정보(가격, 카테고리, 등급 등)를 분석하기 위한 용도로만 사용됩니다.

5. 제3자 공유 및 공개

Sellgent AI는 어떠한 데이터도 제3자와 공유하지 않습니다. 다만 아래의 경우는 예외로 합니다.

  • 법적 요청: 법원 명령, 소환장 등 법적 의무가 있는 경우
  • 서비스 제공자: 서비스 운영에 필수적인 제3자 (데이터 공유 없음)

6. 데이터 보안

  • 모든 데이터는 사용자 브라우저 내에서만 처리됩니다.
  • 네트워크를 통한 데이터 전송이 없습니다.
  • 신용카드, 계좌번호 등 민감한 개인정보는 수집하지 않습니다.

7. 사용자 권리

  • 접근권: 브라우저 저장소에서 직접 확인 가능
  • 수정권: 설정 값 수정 가능
  • 삭제권: 데이터 삭제 또는 확장 프로그램 제거

8. 데이터 보유 기간

  • 로컬 저장 데이터는 사용자가 삭제하거나 확장 프로그램 제거 시 삭제됩니다.
  • 브라우저 캐시 삭제 시 함께 제거됩니다.

9. Cookie 및 추적 기술

Sellgent AI는 다음을 사용하지 않습니다.

  • 개인 추적을 위한 Cookie
  • Google Analytics, Facebook Pixel 등 추적 도구
  • 광고 목적의 추적 기술

10. 아동의 개인정보

본 확장 프로그램은 모든 연령대에서 사용 가능하며 개인정보를 수집하지 않습니다.

11. 정책 변경

본 개인정보 처리방침이 변경될 경우 이 페이지를 통해 공지되며, 변경 사항은 상단의 최종 수정일에 반영됩니다.
